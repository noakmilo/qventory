Need help with CSV format?

Qventory CSV Requirements:

Required:
  • sku - Unique product code
  • title - Item name
Optional (common):
  • item_cost - Purchase price
  • item_price - Selling price
  • supplier - Where you got it
  • listing_date - When listed (YYYY-MM-DD)
Locations:
  • A - Aisle
  • B - Bay
  • S - Shelf
  • C - Container
Marketplace Links:
  • ebay_url
  • amazon_url
  • mercari_url
  • vinted_url
  • + more

Example:

sku,title,item_cost,item_price,supplier,A,B
QV001,Nike Air Max,45.00,89.99,Outlet Store,1,3
QV002,Vintage Camera,12.50,35.00,Thrift Shop,2,1
